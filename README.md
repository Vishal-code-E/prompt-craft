Prompt Craft

A platform for building structured AI prompts with team collaboration.

Build Prompts
Visual editor for creating JSON-formatted prompts. Configure tasks, rules, story elements, and moderation filters. Real-time preview shows your prompt structure as you build.

Work Together
Create workspaces for your team. Add members with different permission levels. Switch between multiple workspaces. Every action is logged for accountability.

Track Changes
All prompts are versioned automatically. View history, compare versions, restore previous states. Never lose work.

Share Securely
Generate links to share prompts. Set expiration dates. Control who can access your work.

Plans Available
Free tier for individuals. Pro and Team plans for advanced features. Secure payment processing.

Start Using

Sign in with Google. Create a workspace. Start building prompts.

Setup

pnpm install
pnpm exec prisma generate
pnpm exec prisma db push
pnpm dev
